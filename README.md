# 💳 Payoo MFS Bank

**Payoo MFS Bank** is a simple mobile banking web application built using **HTML5, CSS3, and raw JavaScript**.
This project simulates basic Mobile Financial Service (MFS) features like adding money, cash out, and checking balance.

It focuses on practicing core JavaScript concepts without using any frameworks or libraries.

---

## 📌 Project Overview

Payoo MFS Bank is a front-end based simulation of a mobile banking system.
Users can perform basic financial operations through an interactive interface.

The project demonstrates:

* DOM manipulation
* Event handling
* Conditional logic
* Basic transaction handling
* Form validation
* Simple UI design

---

## ✨ Features

*  Add Money
*  Cash Out
*  Transaction History (basic)
*  Real-time Balance Update
*  Simple PIN verification logic
*  Responsive layout
*  Interactive buttons and alerts

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout design
* **Vanilla JavaScript (ES6)** –

  * DOM selection
  * Event listeners
  * Functions
  * Conditionals
  * Basic validation logic

## 💻 How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Double-click on `index.html`.
4. The application will open in your browser.

No frameworks, no dependencies — fully built with raw HTML, CSS, and JavaScript.
